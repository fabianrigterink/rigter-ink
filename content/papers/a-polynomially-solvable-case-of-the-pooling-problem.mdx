The pooling problem is NP-hard in general, but special cases arising in practice often have additional structure. We identify a topological condition on the network — a generalisation of being a tree — under which the pooling problem becomes polynomially solvable.

Specifically, we show that when the pool-to-output subgraph has a certain series-parallel structure, the bilinear quality-balance constraints can be linearised exactly. This allows the problem to be solved as a linear program in polynomial time. We provide a combinatorial algorithm to test whether a network satisfies the condition and construct the LP if it does. The result explains why several industrial instances solve easily despite the general hardness of the problem.
