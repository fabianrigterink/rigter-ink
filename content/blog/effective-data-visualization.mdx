A chart is an argument made visual. The best data visualizations don't just show data — they reveal patterns, tell stories, and change minds.

![Colorful data visualization dashboard on a screen](https://images.unsplash.com/photo-1551288049-bebda4e38f71?w=1200&h=600&fit=crop&q=80)

## Start with the question

Before choosing a chart type, ask: what question does this visualization answer? "How has revenue changed over time?" suggests a line chart. "How do categories compare?" suggests a bar chart. "What's the distribution?" suggests a histogram.

The chart type should follow from the question, not the other way around.

## Less is more

Edward Tufte introduced the concept of the **data-ink ratio** — the proportion of a chart's ink that represents actual data. Maximize it by removing:

- Gridlines (or make them very subtle)
- Unnecessary borders and boxes
- 3D effects (always)
- Redundant labels
- Decorative elements

```python
import matplotlib.pyplot as plt
import numpy as np

fig, ax = plt.subplots(figsize=(8, 4))
months = ["Jan", "Feb", "Mar", "Apr", "May", "Jun"]
values = [12, 19, 15, 25, 22, 30]

ax.bar(months, values, color="#007BA7", width=0.6)

# Clean up: remove top and right spines
ax.spines["top"].set_visible(False)
ax.spines["right"].set_visible(False)
ax.spines["left"].set_visible(False)

# Subtle grid, no tick marks
ax.yaxis.grid(True, alpha=0.3)
ax.tick_params(left=False, bottom=False)
ax.set_axisbelow(True)

plt.tight_layout()
plt.savefig("clean_chart.png", dpi=150, bbox_inches="tight")
```

## Color with purpose

Color should encode meaning, not decorate. Use a limited palette and assign colors consistently:

- **Sequential**: for ordered data (light to dark)
- **Diverging**: for data with a meaningful midpoint (e.g., profit/loss)
- **Categorical**: for unordered groups (distinct hues)

Avoid red-green combinations — about 8% of men have red-green color blindness. Test your visualizations with a color blindness simulator.

## Annotation is underrated

The most impactful improvement you can make to any chart is adding clear annotations. Label the key takeaway directly on the chart. Don't make your audience hunt for the insight — point to it.

A chart without context is just shapes. A chart with a clear title, labeled annotations, and a source citation becomes evidence.

## Tell the truth

Truncated y-axes, cherry-picked time ranges, dual y-axes with mismatched scales — these are the tools of deception. Even when unintentional, they erode trust.

Always start bar charts at zero. Always include axis labels. Always cite your data source. The goal is to communicate clearly, not to persuade unfairly.
