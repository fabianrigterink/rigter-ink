Not all data fits neatly into grids or sequences. Social networks, molecular structures, transportation systems — these are naturally represented as graphs. Graph neural networks (GNNs) extend deep learning to this domain.

![Abstract network visualization with interconnected nodes](https://images.unsplash.com/photo-1551288049-bebda4e38f71?w=1200&h=600&fit=crop)

## The message passing framework

Most GNNs follow a simple pattern called **message passing**. At each layer, every node:

1. **Aggregates** messages from its neighbors
2. **Updates** its own representation using the aggregated information

Formally, for node $v$ at layer $k$:

$$
h_v^{(k)} = \text{UPDATE}\left(h_v^{(k-1)}, \text{AGGREGATE}\left(\{h_u^{(k-1)} : u \in \mathcal{N}(v)\}\right)\right)
$$

where $\mathcal{N}(v)$ is the set of neighbors of node $v$.

## A simple GNN in PyTorch Geometric

```python
import torch
import torch.nn.functional as F
from torch_geometric.nn import GCNConv
from torch_geometric.datasets import Planetoid

# Load the Cora citation network
dataset = Planetoid(root="/tmp/Cora", name="Cora")
data = dataset[0]

class GCN(torch.nn.Module):
    def __init__(self, in_channels, hidden_channels, out_channels):
        super().__init__()
        self.conv1 = GCNConv(in_channels, hidden_channels)
        self.conv2 = GCNConv(hidden_channels, out_channels)

    def forward(self, x, edge_index):
        x = self.conv1(x, edge_index)
        x = F.relu(x)
        x = F.dropout(x, p=0.5, training=self.training)
        x = self.conv2(x, edge_index)
        return F.log_softmax(x, dim=1)

model = GCN(dataset.num_features, 16, dataset.num_classes)
```

## The over-smoothing problem

As you stack more GNN layers, node representations converge to the same value — every node ends up looking the same. This is called **over-smoothing**, and it's a fundamental challenge.

Think of it like repeatedly averaging with your neighbors. After enough rounds, everyone's value converges to the global mean. Solutions include residual connections, jumping knowledge networks, and graph transformers.

## Real-world applications

GNNs shine in domains with natural graph structure: drug discovery (predicting molecular properties), recommendation systems (user-item interaction graphs), fraud detection (transaction networks), and traffic prediction (road networks).

The key advantage over traditional approaches is that GNNs can learn to extract features from the graph structure itself, rather than relying on hand-crafted graph statistics like degree centrality or clustering coefficients.
